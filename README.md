# Restaurant POS Simulator

HTML, CSS, JavaScript, jQuery, Bootstrap ve Lottie animasyonları kullanılarak geliştirilmiş mini restoran POS simülasyonu.

## Özellikler

- Ürün listeleme
- Sepete ürün ekleme
- Sepette adet artırma / azaltma / silme
- POS cihazı ekranı simülasyonu
- Birden fazla sipariş kuyruğu
- Sipariş seçerek ödeme alma
- Kart okutma, ödeme alınıyor ve ödeme başarılı akışı
- Lottie payment success animasyonu
- Fiş oluşturma
- Mutfak ekranı
- Mutfak sipariş durumu: Bekliyor / Hazırlanıyor / Hazır
- Toast bildirim sistemi
- LocalStorage ile verileri saklama

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5
- Lottie Web
- LocalStorage
